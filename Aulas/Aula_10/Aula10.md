# Manipulação de arquivo, Decomposição funcional 
## Blocklist de Palavras Pejorativas
    - arquivo de texto (csv) como palavras ou expressoes pejorativas
    - lista para analisar as expressoes pejorativas
    - variável texto (digitada pelo usuário)

Fluxo ou dinâmica do sistema
    1) executar o sistema em terminal CLI (Command line interface)
    2) carregar as palavras pejorativas do arquivo para dentro da lista
    3) receber texto do usuário
    4) analisar se as palavras do texto aparecem na lista de palavras pejorativas
    5) mostrar as palavras do texto que estão na lista, bem como a quantidade delas
